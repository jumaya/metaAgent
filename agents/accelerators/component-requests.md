# Registro de Componentes UI Pendientes

Mantenido por `@nextjs-generator`. Cuando se necesita un componente que no existe
en la biblioteca, se registra aquí en lugar de crearlo inline.

`@component-library-agent` revisa este archivo periódicamente para incorporar
los componentes más solicitados a la biblioteca oficial.

## Formato de registro
```
- [ ] NombreComponente | Solicitado por: frmX | Descripción: para qué se necesita
```
